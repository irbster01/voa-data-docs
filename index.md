## About This Documentation

This documentation explains the structure and purpose of data tables, reports, pipelines, and internal applications used across VOA programs. It's intended to help staff understand where data comes from, how reports are generated, and how to use internal tools.

**Note**: This documentation describes data structures and systems only - no actual client data or sensitive information is included.
